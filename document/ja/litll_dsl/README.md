# Sora DSL ツールキット

Sora DSL ツールキットは、SoraベースのDSLを簡単に作成するためのツール群です。

SoraIDL(Soraインターフェイス記述言語)を使って、データ構造を記述するだけでDSLの作成が終わり、パーサ、バリデータ、シンタックスハイライタ、入力補完を手に入れることができます。

# 機能

Sora DSL ツールキットは、以下の機能

* SoraIDLから、RustまたはHaxeのデータ構造を出力。
